gaussoccR Documentation

Gaussian one-class classifier

Description

Builds a data description based on a target class using parametric Gaussian density estimation. Test objects can be classified as either inliers (label = 0) or outliers (label = 1).

Usage

gaussocc <- function(xtrn, xtst, fracrej = 0.05)

Arguments

xtrn

Training set: a data frame with features as columns. Labels are given in the last column. The labels are dummy labels (= 0) since the training set should consist of only inliers.

xtst

Test set: a data frame with features as columns. Labels are given in the last column. The test set can consist of a mix of inliers and outliers.

fracrej

A scalar specifying the reject fraction to consider as outlier when training.

Value

A list:

l[[1]]

Hard labels for test set: inlier = 0, outlier = 1

l[[2]]

Soft labels for test set: probability of inlier

Author(s)

Jimmy Azar

Examples

l <- gausscirc(ntrn_inlier = 200, ntst_inlier =  100, 
               ntst_outlier = 100, show = TRUE, seed=1)
xtrn <- l[[1]]
xtst <- l[[2]]

out <- gaussocc(xtrn, xtst, fracrej = 0.1)
pred <- out[[1]]
