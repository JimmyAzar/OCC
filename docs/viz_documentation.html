vizR Documentation

Visualize classifier decision boundary

Description

Visualizes the decision boundary of a classifier around the target class. A mesh grid of points is set up, and each grid point is classified.

Usage

viz <- function(data, resolution = 100, classifier, ...)

Arguments

data

(2D) training set: a data frame with features as columns. Labels are given in the last column. The labels are dummy labels since the training set should consist only of inliers.

resolution

Grid resolution (default: 100 points per axis)

classifier

Support for current classifiers: gaussocc, parzenocc, knnocc, rknndocc, kmeansocc, somocc, gmmocc.

Value

Displays a scatter plot of training/test set along with classifier's decision boundary contour.

Author(s)

Jimmy Azar

Examples

l <- boomerang(Ntrn = c(200,200), Ntst = c(100,100),
               ntst_outlier = 200, s=0.7, show =TRUE, seed=1)
xtrn <- l[[1]]
xtst <- l[[2]]

#visualize decision boundary and training set
viz(data = xtrn, classifier = "parzen", h = 1, fracrej = 0.1, resolution = 100)

#add final test objects (inliers and outliers)
shape <- c(20,17) #dot, triangle
points(xtst[,1:2],col="violetred",pch=shape[xtst[,3]+1]) #add test objects
legend("bottomright",legend = c("training inlier","test inlier","test outlier"), 
       col = c("blue","violetred","violetred"), pch = c(20,20,17), cex=0.8)
