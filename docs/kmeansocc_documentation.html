kmeansoccR Documentation

K-means one-class classifier

Description

Builds a data description based on a target class using the k-means algorithm. Test objects can be classified as either inliers (label = 0) or outliers (label = 1).

Usage

kmeansocc <- function(xtrn, xtst, k = 10, fracrej = 0.05, seed=1)

Arguments

xtrn

Training set: a data frame with features as columns. Labels are given in the last column. The labels are dummy labels since the training set should consist of only inliers.

xtst

Test set: a data frame with features as columns. Labels are given in the last column. The test set can consist of a mix of inliers and outliers.

k

A scalar specifying the number of clusters (default k = 10).

fracrej

A scalar specifying the reject fraction to consider as outlier when training (default fracrej = 0.05).

seed

A seed value for saving the state of randomness.

Value

A list:

l[[1]]

Hard labels for test set: inlier = 0, outlier = 1

l[[2]]

Cluster labels for training set (1,...,k)

l[[3]]

Cluster centers (dim: k x p)

Author(s)

Examples

l <- boomerang(Ntrn = c(200,200), Ntst = c(100,100), 
               ntst_outlier = 200, s=0.7, show =TRUE, seed=1)
xtrn <- l[[1]]
xtst <- l[[2]]

#Visualize cluster centers
k = 20 #number of clusters
l <- kmeansocc(xtrn, xtst, k=k, fracrej = 0.05, seed = 1)
L <- l[[1]]
clusterId <- l[[2]]
centroids <- l[[3]]

plot(xtrn[,1],xtrn[,2],pch=19,col=clusterId)
points(centroids[,1],centroids[,2],pch=4,col="magenta",lwd=5)
