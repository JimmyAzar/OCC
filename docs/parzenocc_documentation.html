parzenoccR Documentation

Parzen one-class classifier

Description

Builds a data description based on a target class using Parzen density estimation. Test objects can be classified as either inliers (label = 0) or outliers (label = 1).

Usage

parzenocc(xtrn,xtst,h=seq(0.01,10,length.out=50),fracrej=0.05)

Arguments

xtrn

Training set: a data frame with features as columns. Labels are given in the last column. The labels are dummy labels since the training set should consist of only inliers.

xtst

Test set: a data frame with features as columns. Labels are given in the last column. The test set can consist of a mix of inliers and outliers.

h

A scalar or a numeric vector specifying the width of the Gaussian kernel for density estimation.

fracrej

A scalar specifying the reject fraction to consider as outlier when training.

Value

Ltst

Hard labels for test set: inlier = 0, outlier = 1

Ptst

Soft labels for test set: probability of inlier

h

Optimal h selected if a numeric vector was supplied for the parameter

Author(s)

Examples

l <- boomerang(Ntrn = c(200,200), Ntst = c(100,100),
               ntst_outlier = 200, s=0.7, show =TRUE, seed=1)
xtrn <- l[[1]]
xtst <- l[[2]]

out <- parzenocc(xtrn, xtst, h=1, fracrej = 0.1)
pred <- out[[1]]
