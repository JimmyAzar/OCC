somoccR Documentation

Self-organizing map one-class classifier

Description

Builds a data description based on a target class using a self-organizing map. Test objects can be classified as either inliers (label = 0) or outliers (label = 1).

Usage

somocc <- function(xtrn, xtst, gridSize = 4, fracrej = 0.05, seed=1)

Arguments

xtrn

Training set: a data frame with features as columns. Labels are given in the last column. The labels are dummy labels since the training set should consist of only inliers.

xtst

Test set: a data frame with features as columns. Labels are given in the last column. The test set can consist of a mix of inliers and outliers.

gridSize

A scalar specifying a square grid size.

fracrej

A scalar specifying the reject fraction to consider as outlier when training.

seed

A seed value for saving the state of randomness.

Value

A list:

l[[1]]

Hard labels for test set: inlier = 0, outlier = 1

l[[2]]

SOM centroids (dim: k x p)

Author(s)

Examples

l <- boomerang(Ntrn = c(200,200), Ntst = c(100,100),
               ntst_outlier = 200, s=0.7, show =TRUE, seed=1)
xtrn <- l[[1]]
xtst <- l[[2]]

#Visualize cluster centers
gridSize <- 5
l <- somocc(xtrn, xtst, gridSize = gridSize, fracrej = 0.05, seed = 1)
L <- l[[1]]
centroids <- l[[2]]

plot(xtrn[,1],xtrn[,2],pch=19, xlab="Feature 1",ylab="Feature 2")
points(centroids[,1],centroids[,2],pch=4,col="magenta",lwd=5)
