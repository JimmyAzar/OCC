gmmoccR Documentation

Gaussian mixture model one-class classifier

Description

Builds a data description based on a target class using a Gaussian mixture model density estimation. Test objects can be classified as either inliers (label = 0) or outliers (label = 1).

Usage

gmmocc <- function(xtrn, xtst, k = 6, fracrej = 0.05, seed=1)

Arguments

xtrn

Training set: a data frame with features as columns. Labels are given in the last column. The labels are dummy labels since the training set should consist of only inliers.

xtst

Test set: a data frame with features as columns. Labels are given in the last column. The test set can consist of a mix of inliers and outliers.

k

A scalar specifying the number of Gaussian components.

fracrej

A scalar specifying the reject fraction to consider as outlier when training.

fracrej

A scalar specifying the reject fraction to consider as outlier when training.

seed

A seed value for saving the state of randomness.

Value

A list:

l[[1]]

Hard labels for test set: inlier = 0, outlier = 1

l[[2]]

Soft labels for test set: probability of inlier

l[[3]]

Gaussian mixture model object

Author(s)

Jimmy Azar

l <- gausscirc(ntrn_inlier = 200, ntst_inlier =  100,
