boomerangR Documentation

Boomerang-shaped dataset

Description

Generates a boomerang-shaped dataset with two branches in 2D as training set, in addition to a test set consisting of inliers from the same boomerang distributions as well as outliers from a uniform distribution. Inliers have label = 0 and outliers have label = 1.

Usage

boomerang <- function(Ntrn=c(50,50), Ntst = c(20,20), ntst_outlier=100, s=1, 
                      show=TRUE,seed=1)

Arguments

Ntrn

Numeric vector specifying the number of inliers per branch in the training set.

Ntst

Numeric vector specifying the number of inliers per branch in the test set.

ntst_outlier

Number of outliers in the test set drawn from a uniform distribution.

s

A scalar parameter that controls the width of the branches.

show

If TRUE, a scatter plot is generated showing the training/test set.

seed

A seed value for saving the state of randomness.

Value

A list:

l[[1]]

Training set: a data frame with labels as last column

l[[2]]

Test set: a data frame with labels as last column

Author(s)

Examples

l <- boomerang(Ntrn = c(200,200), Ntst = c(100,100), ntst_outlier = 200, s=0.7, 
               show =TRUE, seed=1)
xtrn <- l[[1]]
xtst <- l[[2]]
